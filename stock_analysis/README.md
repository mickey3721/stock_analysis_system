## 三、核心分析模块

### 3.1 均线系统

**算法公式（EMA）：**
```
EMA(x, n) = α × x + (1-α) × EMA(x, n-1)，α = 2/(n+1)
```

| 周期 | 参数 | 用途 |
|------|------|------|
| EMA5 | 5日 | 短期攻防线 |
| EMA26 | 26日 | 中期均线 |
| EMA89 | 89日 | 长期均线 |
| EMA144 | 144日 | 超长趋势 |

---

### 3.2 徐小明多空通道

**算法公式：**

```
┌─────────────────────────────────────────────────────────────┐
│              短线通道（25日）                                │
├─────────────────────────────────────────────────────────────┤
│  上轨 = EMA(H, 25)                                         │
│  下轨 = EMA(L, 25)                                         │
│                                                             │
│              长线通道（89日）                                │
├─────────────────────────────────────────────────────────────┤
│  上轨 = EMA(H, 89)                                         │
│  下轨 = EMA(L, 89)                                         │
│                                                             │
│  其中：                                                     │
│  - H = 最高价                                               │
│  - L = 最低价                                               │
│  - EMA(x, n) = α × x + (1-α) × EMA(x, n-1)，α = 2/(n+1)  │
└─────────────────────────────────────────────────────────────┘
```

**颜色信号判定：**
```
IF (当前价 > 上轨) THEN 突破信号
ELSE IF (当前价 < 下轨) THEN 跌破信号
ELSE IF (上轨 > 上轨[1]) THEN 红色（多头）
ELSE THEN 绿色（空头）
```

| 颜色组合 | 信号 | 说明 |
|----------|------|------|
| 红色+紫色 | 强烈做多 | 长短线共振向上 |
| 绿色+蓝色 | 强烈做空 | 长短线共振向下 |
| 绿色+紫色 | 谨慎做多 | 长线多头，短线回调 |
| 红色+蓝色 | 谨慎观望 | 长线空头，短线反弹 |

---

### 3.3 九转序列（TD序列）

**算法公式（TD序列规则）：**
```
买序列（TD买入计数）：收盘价 >= 前第4根收盘价 时计数+1，否则重置为1
卖序列（TD卖出计数）：收盘价 <= 前第4根收盘价 时计数+1，否则重置为1
```

**关键点：允许中间有反向K线，只要满足与前第4根的比较条件即可计数**

| 序列位置 | 信号 |
|----------|------|
| 序列6/7/8 | 预警信号 |
| 序列9 | 高概率反转点 |

---

### 3.4 MACD定量结构

**算法公式：**

```
1. MACD计算：
   DIF = EMA(close, 12) - EMA(close, 26)
   DEA = EMA(DIF, 9)
   MACD = (DIF - DEA) × 2

2. 钝化判定（120周期窗口）：
   顶部钝化 = 价格创新高(120周期内) AND DIF未创新高(120周期内)
   底部钝化 = 价格新低(120周期内) AND DIF未新低(120周期内)

3. 定量级别：
   75%结构 = 钝化状态 AND DIF发生转向
   100%结构 = 75%结构 AND DIF与DEA交叉
```

| 结构级别 | 条件 | 可靠性 |
|----------|------|--------|
| 钝化 | 价格与DIF背离 | 预警 |
| 75% | 钝化 + DIF转向 | 高 |
| 100% | 75% + 金叉/死叉 | 最高 |

### 3.5 缠论分析（简化版，参考czsc）

```
分型：顶分型/底分型
笔：顶分型和底分型之间，至少有一根独立K线
     - 上升笔：由底分型开始，到顶分型结束
     - 下降笔：由顶分型开始，到底分型结束
线段：3笔重叠
中枢：线段重叠区域
背驰：价格创新高/低但MACD面积缩小

注意：此为简化版缠论概念示意，分析结果仅供参考
```

## 四、风控模块

### 4.1 风控配置

```python
RISK_CONFIG = {
    "stop_loss": {"pct": 0.07},           # 止损：-7%
    "take_profit": {
        "fixed_pct": 0.15,                # 止盈：+15%
        "trailing_pct": 0.05,             # 回落5%止盈
        "start_pct": 0.10
    },
    "drawdown": {
        "max_drawdown": 0.15,             # 最大回撤：-15%
        "cooling_days": 5
    },
    "position": {
        "max_stocks": 3,
        "max_single": 0.30
    }
}
```

### 4.2 风控逻辑

```
触发条件：
├── 止损：当前价 ≤ 买入价 × 93%
├── 止盈1：盈利 ≥ 15% → 全部卖出
├── 止盈2：盈利 ≥ 10% 且回落 ≥ 5% → 跟踪卖出
├── 熔断：总资金回撤 ≥ 15% → 清仓休息5天
└── 减仓：单只回撤 ≥ 10% → 减仓50%
```

## 七、数据库设计

### 7.1 K线数据表

```sql
CREATE TABLE kline (
    id INTEGER PRIMARY KEY,
    symbol TEXT NOT NULL,
    trade_date TEXT NOT NULL,
    open REAL, high REAL, low REAL, close REAL,
    volume REAL, amount REAL,
    period TEXT,
    created_at TIMESTAMP
);
```

### 7.2 综合分析结果表

```sql
CREATE TABLE analysis_result (
    id INTEGER PRIMARY KEY,
    symbol TEXT NOT NULL,
    period TEXT NOT NULL,
    trade_date TEXT,

    -- 均线
    ma5 REAL, ma26 REAL, ma89 REAL, ma144 REAL,

    -- 多空通道
    short_upper REAL, short_lower REAL,
    long_upper REAL, long_lower REAL,
    short_status TEXT, long_status TEXT,
    channel_signal TEXT,

    -- 九转序列
    seq9_buy INTEGER, seq9_sell INTEGER,

    -- MACD
    dif REAL, dea REAL, macd REAL,
    macd_structure TEXT,

    -- 支撑/阻力
    support_levels TEXT, resistance_levels TEXT,

    -- 综合信号
    signal TEXT, confidence REAL,
    analysis_time TIMESTAMP
);
```

CREATE TABLE trades (

## 九、核心算法汇总

| 指标 | 周期/参数 | 公式 | 应用场景/信号含义 |
|------|-----------|------|-------------------|
| EMA5 | 5日 | α×C+(1-α)×EMA，α=2/6 | 短期趋势、动态支撑 |
| EMA26 | 26日 | α×C+(1-α)×EMA，α=2/27 | 中期趋势 |
| EMA89 | 89日 | α×C+(1-α)×EMA，α=2/90 | 长期趋势 |
| EMA144 | 144日 | α×C+(1-α)×EMA，α=2/145 | 超长趋势 |
| 短线通道 | 25日 | EMA(H,25)/EMA(L,25) | 短期多空、强弱判断 |
| 长线通道 | 89日 | EMA(H,89)/EMA(L,89) | 长期多空、趋势判断 |
| DIF | 12/26 | EMA(C,12)-EMA(C,26) | 动能变化 |
| DEA | 9 | EMA(DIF,9) | 信号线 |
| 九转序列 | TD规则 | 与前第4根比较计数 | 择时信号、高9反转 |
| MACD钝化 | 120周期 | 窗口内创新高/低判定 | 顶底背离、结构形成 |
