## 2026-03-30 (续)

### 任务：多周期支持

#### 完成工作

1. **扩展分钟数据获取**
   - 添加指数分钟数据: `_get_index_minute()`
   - 添加个股分钟数据: `_get_stock_minute()`
   - 支持周期: 5/15/30/60/120分钟

2. **添加实时行情接口**
   - `get_realtime()` - 获取实时行情
   - `get_tick()` - 获取分时数据

3. **更新周期配置**
   - config.py 添加 PERIOD_CONFIG
   - 各周期对应不同均线参数

4. **更新主程序**
   - 支持命令行参数 `-p/--period`
   - 支持 `-s/--symbol` 指定标的
   - 支持 `-a/--all` 分析所有周期

#### 使用方法
```bash
# 日线分析（默认）
python main.py

# 60分钟分析
python main.py -p 60min

# 5分钟分析
python main.py -p 5min

# 指定标的
python main.py -s 600176.SH

# 所有周期
python main.py -a

# 组合使用
python main.py -a -s 600176.SH
```

#### 测试结果
```
60min 600176.SH: 126条 ✅
5min 000001.SH: 1512条 ✅
```
